# Echodesk Ecommerce Template

A modern, customizable ecommerce storefront template built with Next.js 15, shadcn/ui, and Tailwind CSS.

## Features

- **Dynamic Theme**: Colors, logo, and branding customizable via environment variables
- **Multi-Currency Support**: GEL, USD, EUR, and more
- **Multi-Language Ready**: Built-in internationalization support
- **shadcn/ui Components**: Beautiful, accessible UI components
- **Responsive Design**: Mobile-first approach
- **SEO Optimized**: Dynamic metadata and Open Graph tags

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Backend API running (Echodesk CRM)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Configure environment variables:
```bash
cp .env.template .env.local
```

3. Update `.env.local` with your tenant-specific values:
```env
NEXT_PUBLIC_TENANT_ID=your-tenant
NEXT_PUBLIC_API_URL=https://your-tenant.api.echodesk.ge
NEXT_PUBLIC_STORE_NAME=Your Store Name
NEXT_PUBLIC_PRIMARY_COLOR=221 83% 53%
# ... other variables
```

4. Run development server:
```bash
npm run dev
```

5. Build for production:
```bash
npm run build
npm start
```

## Environment Variables

See `.env.template` for all available configuration options:

- **Tenant Configuration**: API URL, tenant ID
- **Store Branding**: Name, description, logo
- **Theme Colors**: Primary, secondary, accent colors (HSL format)
- **Locale Settings**: Currency, language
- **Features**: Enable/disable wishlist, reviews, compare
- **Social Links**: Facebook, Instagram, Twitter
- **Contact Info**: Email, phone, address

## Customization

### Theme Colors

Colors use HSL format for Tailwind CSS compatibility:
```
Primary: 221 83% 53% (Blue)
Secondary: 215 16% 47% (Slate)
Accent: 142 71% 45% (Green)
```

### Logo

Place your logo in `/public/logo.png` or update `NEXT_PUBLIC_STORE_LOGO` env variable.

### Components

All UI components are in `src/components/ui/` and can be customized as needed.

## API Integration

The template expects the Echodesk CRM API endpoints:

- `GET /api/ecommerce/client/products/` - List products
- `GET /api/ecommerce/client/attributes/` - Get filterable attributes
- `POST /api/ecommerce/clients/login/` - Client authentication
- `POST /api/ecommerce/clients/register/` - Client registration
- And more...
